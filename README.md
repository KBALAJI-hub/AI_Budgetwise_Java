# Finance Tracker & Budgeting App

## Setup

1. **Backend**:
   - `cd server`
   - `npm install`
   - Configure `.env`
   - `npx prisma migrate dev`
   - `npm start`

2. **Frontend**:
   - `cd client`
   - `npm install`
   - `npm run dev`

## Features
- JWT Authentication
- Budget Tracking
- Analytics with Charts
- Admin Dashboard (`/admin`)
- Swagger Docs (`/api-docs`)
- Dockerized Backend
